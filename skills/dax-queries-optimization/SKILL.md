# Skill de Agente: DAX-QUERIES - Otimização e performance

## Descrição Operacional
Esta skill instrui o agente como lidar com a tarefa de **Otimização e performance** para a tecnologia **dax-queries**.

## Diretrizes de Execução
1. **Identificar Requisitos:** Verifique dependências necessárias antes de iniciar.
2. **Seguir Padrões:** Use a melhor prática oficial documentada para dax-queries.
3. **Validar:** Execute verificações de sanidade e segurança após a implementação.
4. **Registrar:** Mantenha logs claros das alterações realizadas para auditoria.

## Dicas Técnicas Específicas
- Para dax-queries, sempre certifique-se de usar caminhos seguros e bibliotecas atualizadas.
- Evite alocação desnecessária de memória.
- Limite as permissões de acesso ao menor privilégio possível.
